What is DOCTYPE statement?
A DOCTYPE is a required preamble that tells the browser that this is html.
How do you use: <ol>;<ul>;<dl>
<ol>:Used to create a numbered list. You have to use <li> for each new item.
<ul>:Used to create a list using bullet points.You have to use <li> for each new item.
<dl>:Used to create a list with descriptions. First you use <dt> to state the term, then you use <dd> to describe the term.
Difference between text editor and word processor?
A text editor is used to write and edit plain texts. A word processor lets you edit texts using other functions like formatting.
What is the client server model?
The client server model is a computing architecture where the clients request services and servers provide for them.
Provide a description and definition for the following network protocols.
FTP:This is a method for moving files between computers on a network. It allows users to upload and download files to and from servers.
HTTP:This is the foundation of the world wide web, it allows your web browser to request information from a server and display web pages.
TCP/IP: This is the suite of communication protocols used to connect network devices on the internet. TCP ensures reliability data delivery, while IP takes care of addressing and routing of data packets.
